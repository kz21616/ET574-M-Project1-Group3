Project Tasks
1. Menu System (main.py): Build a menu loop with options:
- Add Book
- View Reading List
- View Summary
- Exit
2. Add Book (tracker.py): Prompt for book title, time spent reading, and status (Finished/In Progress). Append to
parallel lists in `data.py`
3. View Reading List: Print each entry in a formatted list
4. Summary Report: Show total time spent reading, number of books, count of Finished vs In Progress, and the
book with most time spent
